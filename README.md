# docker-java-examples
Java repo containing example code and docker files to containerize an application

This repo contains code for some of my medium articles, checkout my articles at [@praneethbonagiri.pb](https://medium.com/@praneethbonagiri.pb)

- branch docker_fundamentals contains code for [Docker for Beginners](https://medium.com/@praneethbonagiri.pb/docker-fundamentals-containerizing-a-simple-java-application-208e546d506b) medium article
- branch docker_volumes contains code for [Understanding Docker Volumes](https://medium.com/@praneethbonagiri.pb/managing-data-in-docker-volumes-and-persistence-47a3ac7bcc07) medium article
- branch docker_networking_and_compose contains code for [Docker networking and docker compose](https://medium.com/@praneethbonagiri.pb/docker-networking-docker-compose-connecting-containers-1a4f51993248) medium article
